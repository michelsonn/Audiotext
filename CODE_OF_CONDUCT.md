# Кодекс поведения

Участники проекта должны общаться уважительно и по существу.

Недопустимы оскорбления, травля, дискриминация, публикация чужих персональных данных и намеренное нарушение работы проекта.

Обсуждайте код и решения, а не личность автора. Критика должна содержать понятное техническое обоснование и, по возможности, способ улучшения.
