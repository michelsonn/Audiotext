# Security Policy

Пожалуйста, не публикуйте сведения об уязвимости в открытом issue до исправления.

До появления отдельного приватного канала связи подготовьте минимальный воспроизводимый пример без реальных аудиозаписей, персональных данных, токенов и локальных путей пользователя.

Проект не должен собирать или отправлять пользовательские аудиозаписи. Обращение к интернету ожидается при загрузке моделей и зависимостей.
