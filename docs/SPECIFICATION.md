# Audiotext — спецификация проекта

Версия документа: 1.0.6  
Продукт: **Audiotext**  
Описание: **Многопоточный транскрибатор**  
Авторские права: **© 2026 Mikhail Zuev**

## 1. Назначение

Audiotext — Windows-приложение с графическим интерфейсом для быстрой локальной транскрибации аудио на NVIDIA GPU. Оно предназначено как для разовой обработки файлов через очередь и drag-and-drop, так и для массовой рекурсивной обработки папок.

Основной проверенный сценарий — русскоязычные записи телефонных разговоров с использованием Faster Whisper и модели Large V3 Turbo.

## 2. Технологический стек

- Python 3.11/3.12 x64;
- PySide6 — графический интерфейс;
- faster-whisper;
- CTranslate2;
- NVIDIA CUDA;
- Hugging Face Hub — загрузка моделей;
- PyInstaller — будущая portable EXE-сборка.

CPU и AMD/Radeon в версии 1.x не являются поддерживаемыми рабочими режимами.

## 3. Основные режимы

### Вкладка «Файлы»

- добавление одного или нескольких аудиофайлов;
- добавление папки в очередь;
- drag-and-drop файлов и папок;
- очередь с колонками «Файл», «Статус», «Прогресс», «Результат»;
- удаление выбранных элементов;
- очистка завершённых;
- мягкая остановка после текущих заданий;
- двойной щелчок по строке открывает предпросмотр TXT, SRT и JSON.

### Вкладка «Папки»

- рекурсивный поиск аудио;
- сохранение рядом с аудио;
- сохранение в отдельную папку с зеркальной структурой;
- пропуск уже готовых результатов;
- добавление найденных файлов в общую очередь.

## 4. Выходные файлы

Для каждого аудиофайла создаются:

- `.txt` — сплошная транскрипция;
- `.srt` — транскрипция с таймкодами;
- `.segments.json` — служебные сегменты и метаданные.

JSON нужен для восстановления TXT/SRT и технической диагностики. Он не исправляет ошибки распознавания или VAD.

Запись результатов должна быть безопасной: временный файл создаётся до окончательного переименования.

## 5. Модели

В интерфейсе показываются понятные названия:

- **Large V3 Turbo — рекомендуется**  
  `mobiuslabsgmbh/faster-whisper-large-v3-turbo`
- **Large V3 — максимальная точность**  
  `Systran/faster-whisper-large-v3`
- **Medium — быстрее**  
  `medium`
- **Small — очень быстро**  
  `small`
- **Другая модель…** — ручной ввод Hugging Face ID.

Технический идентификатор скрыт за понятной подписью. Локально выбранная папка модели имеет приоритет над моделью из списка.

Модели хранятся в portable-структуре:

```text
Audiotext\
  models\
```

## 6. Данные программы

Portable-данные хранятся рядом с программой:

```text
Audiotext\
  data\
    settings.ini
    queue.json
  models\
```

- `settings.ini` — последние настройки и размеры колонок;
- `queue.json` — сохранённая очередь;
- при незавершённой очереди программа спрашивает, продолжить ли её.

Старый путь `%LOCALAPPDATA%\Audiotext` используется только для одноразовой миграции настроек. Большие модели автоматически не копируются.

## 7. Производительность

- число воркеров: 1–32;
- рабочее значение зависит от объёма VRAM и длины файлов;
- проверено, что несколько воркеров дают существенный прирост относительно Buzz;
- VAD включён по умолчанию;
- во время обработки блокируется автоматический переход Windows в сон.

Проверенная конфигурация VAD:

```text
threshold: 0.5
min_silence_duration_ms: 2000
speech_pad_ms: 400
```

## 8. Интерфейс

- светлая тема;
- розово-малиновый акцент из фирменной иконки;
- одно главное окно;
- вкладки «Файлы» и «Папки»;
- кнопка «Настройки» справа от вкладок;
- отдельное системное меню не используется;
- поддержка русского и английского интерфейса;
- сворачивание в системный трей;
- системное уведомление после завершения;
- ширина колонок таблицы регулируется и сохраняется.

## 9. Диагностика

Раздел диагностики показывает:

- версию Audiotext;
- версию Python;
- путь к исполняемому файлу;
- пути `data` и `models`;
- сведения `nvidia-smi`;
- версию CTranslate2;
- количество доступных CUDA-устройств;
- версию faster-whisper.

Отчёт можно экспортировать в TXT.

## 10. Поддерживаемые форматы

Перечень берётся из `engine.SUPPORTED_AUDIO_EXTENSIONS`. Целевой набор включает распространённые форматы: WAV, MP3, FLAC, M4A и другие, поддерживаемые текущим аудиодекодером Faster Whisper/PyAV.

## 11. Ограничения версии 1.x

- только NVIDIA/CUDA;
- установщик пока не создан;
- автоматическая проверка обновлений архитектурно запланирована, но источник обновлений не задан;
- EXE должен распространяться всей portable-папкой, а не одним файлом;
- сборка Windows EXE выполняется на Windows.
